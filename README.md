# 🤖 J.A.R.V.I.S — Yapay Zekâ Kişisel Asistan

**Just A Rather Very Intelligent System**

Yüz tanıma, sesli komut, hava durumu, haber, YouTube indirme ve çok daha fazlasını sunan yapay zekâ destekli kişisel masaüstü asistanınız.

> **Erum Tech Core** tarafından geliştirilmiştir.

---

## 🚀 Özellikler

- 👁️ **Yüz Tanıma** — LBPH algoritması ile optik yüz tanıma
- 🎤 **Sesli Komut** — Doğal dil ile sesli kontrol
- 🌤️ **Hava Durumu** — Konum tabanlı anlık hava raporu
- 📰 **Haber Bülteni** — API ile güncel haberler
- 📺 **YouTube İndirici** — Video indirme
- 🔍 **Google & YouTube Arama** — Sesli arama
- 📖 **Akıllı Sözlük** — Otomatik yazım düzeltme ile sözlük
- 📧 **E-posta Gönderimi** — SMTP ile e-posta
- 🗺️ **Harita & Konum** — Google Haritalar entegrasyonu
- 💻 **Sistem Bilgisi** — CPU, pil ve sistem durumu
- 📸 **Ekran Görüntüsü** — Sesli komutla ekran görüntüsü
- 😂 **Eğlence Modu** — Şaka ve fıkralar

## 📦 Kurulum

```bash
# Projeyi klonlayın
git clone https://github.com/DevKursat/J.A.R.V.I.S.git

# Masaüstü uygulama klasörüne gidin
cd J.A.R.V.I.S/desktop-app

# Bağımlılıkları kurun
pip install -r requirements.txt

# Çalıştırın
python jarvis.py
```

## 🔧 Gereksinimler

- Python 3.8+
- Mikrofon (sesli komutlar için)
- Web kamerası (yüz tanıma için)
- İnternet bağlantısı
